# ⚽ Tesouraria da Pelada

Aplicação web mobile para controle financeiro de um time de futebol.

## Funcionalidades

- Cadastro de jogadores
- Controle de mensalidade
- Caixa com entradas e saídas
- Subtotais por Pix e Dinheiro
- Histórico mensal
- Reset de mês
- Área administrativa com senha

## Tecnologias

- HTML
- CSS
- JavaScript
- LocalStorage
